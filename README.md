# PowerofBI.IBCS - IBCS-guided data visualization in your Power BI reports

Read more at https://www.powerofbi.org/powerofbi-ibcs-user-defined-functions/.

This repository is for development only. It may contain broken TMDL code or outdated documentation. Use [DAX Lib](https://daxlib.org/package/PowerofBI.IBCS/) to download the most recent stable version of the PowerofBI.IBCS package.

If you have any problems or proposals regarding the IBCS visualizations generated by this package, the UDF functions, or the documentation in the package, feel free to create an [issue](https://github.com/avatorl/dax-udf-svg-ibcs/issues) or pull request.
